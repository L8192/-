## 四、技术架构

```
知识库系统/
├── knowledge_base.json   # 结构化知识库（27条）
├── index.html            # AI问答前端页面
├── server.py             # Flask后端（检索+API代理）
├── requirements.txt      # Python依赖
├── 知识库资料清单.md      # 本文件
└── mermaid_png/          # 流程图渲染缓存（预留）
```

- **模型**：deepseek-v4-pro
- **检索方式**：中文关键词分词匹配（Top 5）
- **响应模式**：仅基于知识库内容回答
